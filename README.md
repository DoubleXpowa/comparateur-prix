## 🚀 Installation sur mobile

### Option 1 : Hébergement en ligne (RECOMMANDÉ)

Pour installer l'application sur votre téléphone, vous devez d'abord l'héberger en ligne. Voici les solutions gratuites :

#### **A. GitHub Pages (gratuit, facile)**

1. Créez un compte sur https://github.com
2. Créez un nouveau repository (ex: "comparateur-prix")
3. Uploadez TOUS les fichiers (HTML, JSON, JS, PNG)
4. Allez dans Settings → Pages
5. Activez GitHub Pages
6. Vous obtenez un lien : `https://votrenom.github.io/comparateur-prix/comparateur_prix.html`
7. Ouvrez ce lien sur votre mobile
8. Suivez les instructions ci-dessous pour installer

#### **B. Netlify Drop (le plus simple)**

1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez TOUS les fichiers
3. Vous obtenez un lien instantané
4. Ouvrez-le sur votre mobile

#### **C. Vercel (gratuit)**

1. Créez un compte sur https://vercel.com
2. Uploadez les fichiers
3. Déployez en un clic

### 📱 Installation sur Android (Chrome)

1. Ouvrez le lien de votre application dans **Chrome**
2. Un bandeau apparaît en bas : **"Installer l'application ?"**
3. Appuyez sur **"Installer"**
4. L'icône apparaît sur votre écran d'accueil ! 🎉

**Alternative :**
- Menu ⋮ → **"Ajouter à l'écran d'accueil"**

---

### 📱 Installation sur iPhone/iPad (Safari)

1. Ouvrez le lien dans **Safari**
2. Appuyez sur le bouton **Partager** 📤 (en bas)
3. Sélectionnez **"Sur l'écran d'accueil"**
4. Appuyez sur **"Ajouter"**
5. L'icône apparaît sur votre écran d'accueil ! 🎉

## 💡 Utilisation

### Touches rapides :
- **Entrée** ⌨️ : Ajouter une nouvelle ligne
- **✏️** : Cliquer sur les titres pour les renommer
- **🗑️** : Effacer toute une section

### Offres promo :
- **1+1@50%** : 1 acheté, le 2ème à -50%
- **1+1@25%** : 1 acheté, le 2ème à -25%
- **2+1** : 2 achetés, 1 offert
- Fonctionne avec **tous les % de 1 à 99** !

### Réductions :
- **Réd. (€)** : Montant fixe (ex: 0.50)
- **Réd. (%)** : Pourcentage direct (ex: 20 pour 20%)

## 🔧 Test local (sans hébergement)

Si vous voulez tester localement avant de déployer :

1. Installez Python : https://www.python.org/downloads/
2. Ouvrez un terminal dans le dossier des fichiers
3. Lancez : `python -m http.server 8000`
4. Ouvrez : `http://localhost:8000/comparateur_prix.html`

⚠️ Note : L'installation PWA ne fonctionnera que sur HTTPS (donc en ligne)
